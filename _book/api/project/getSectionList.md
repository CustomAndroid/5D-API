## 获取标段

> 用于获取项目标段

### 1. 请求说明

> 请求方式：POST
>
> 请求URL ：`/api/AppApi/getSectionList`

### 2. 请求参数

body:

| 字段       | 字段类型 | 必填 | 字段说明 |
| ---------- | :------: | :--: | -------- |
| **userId** |  String  |  Y   | 用户ID   |

示例:

```json
{
  "userId": "496"
}
```

### 3. 返回参数

| 字段      | 字段类型 | 字段说明 |
| --------- | :------: | -------- |
| **id**    |  string  | 节点ID   |
| **pId**   |  String  | 父ID     |
| **value** |  String  | 名称     |

      "id": "600",
